This plugin gives you a professional way to share your files through wordpress.

== Description ==

This plugin gives you a professional way to share your files through wordpress. After installation you will find SharePro icon on wordpress editor. By clicking SharePro icon a modal form will show with media list, by clicking the any file a shortcode with file id will be inserted into your post/page [sharepro file="123"]. You can use shortcodes manually wherever your site allows shortcodes. By default SharePro has 2 access to downloads: first is for free access and second is for pro access. To gain pro access visitor must be logged in. By default there is not set any special role as pro access but if you want in plugin options you can check the checkbox, to force users to request pro access by email from website administrator.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
== Frequently Asked Questions ==

A. Where are files stored?
B. Can users upload files?

A. Files are stored where your wordpress is hosted.
B. No the plugin is made to help admin of site to share files more professionally.
